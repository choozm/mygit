JCALC - Standard and Scientific Calculator

1) OVERVIEW:

V0.1: alpha version

      Standard Calculator only (+, -, /, *)
 
      This version is operationnal, but some minor bug still exist.
      Use the CLEAR button if calcul go wrong (NaN, division by 0,...)	

2) INSTALLATION:

	- Extract the jcalc-alpha-0-1.zip in a folder (e.g. C:\JCALC)

	- Make sure you have the Java runtime environnement (free download from www.java.sun.com)

 	- Edit the text file "run.bat", change the path of "java" to the correct one

	- execute "run.bat" file to start the application
